# Eye Guard

Modern eye-rest reminder for Linux built with PyQt6.

## Features

- 20-20-20 eye reminders
- Fullscreen semi-transparent overlay
- System tray support
- Adjustable reminder timing
- Auto-start compatible
- Pop!_OS inspired UI
